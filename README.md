## HackHeartとは
ハートを叩き割ることをイメージしたアプリです。このアプリは、ピニャータをモチーフとしており、ユーザーがスマホをカッコよく振ることで、そのカッコ良さを競うことができます。また、そのスコアに合わしてスコアの木が作成されます。

## Technology Stack
- Flutter
  - [sensors_plus](https://pub.dev/packages/sensors_plus)
